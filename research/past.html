Past Research Projects

A collection of earlier academic and technical projects that established the foundation for my current work, highlighting the skills, methods, and research directions that continue to influence my development today.

Waksman Student Scholars Program
Field: Molecular Biology & Bioinformatics

Conducted DNA sequence analysis as part of the Waksman Student Scholars Program in collaboration with Rutgers University. Utilized NCBI BLAST tools and bioinformatics databases to identify novel gene sequences from environmental DNA samples, contributing verified data entries to GenBank.

This project provided early exposure to experimental genomics, data validation workflows, and the scientific publication process.

Game Development 3SP Project
Field: Interactive Systems & Software Design

Designed and developed a 2D game using the Unity engine as part of the 3SP initiative, focusing on gameplay mechanics, object collision, and dynamic scene management. Implemented object-oriented programming principles and efficient C# scripting to create responsive and scalable game interactions.

This experience strengthened my applied understanding of software architecture, event systems, and user engagement design.
